CSS
1. Các cách CSS trong HTML
- trong thẻ HTML
- trong file HTML
- ngoài file HTML
2. 4 cách tương tác cơ bản với HTML
- với tên ID của thẻ HTML
- với tên class của thẻ HTML
- với tên thẻ HTML
- với tên thuộc tính và giá trị thuộc tính của thẻ HTML
3. Một số bộ chọn (tương tác) với HTML nâng cao
-  tag01 tag02
-  tag01 > tag02
-  tag01~tag02
-  tag01+tag02
-  tag01,tag02
-  :hover, last-child,....
4. Nguyên tắc cốt lõi khi CSS
- CSS viết sau sẽ được ưu tiên sử dụng
- !important -> tên_id -> tên_thuộc_tính(attribute) và giá_trị -> class -> tên thẻ
- CSS càng viết chi tiết hoặc càng gần thẻ HTML thì càng được ưu tiên
5. Các thuộc tính và giá trị thuộc tính của CSS
- font
- display: flex, none, block, grid
- color
- background
- margin/padding
- border
- border-radius
- with, height
- left, right, top, bottom
- text
- postion, animation,....

Tổng kết bài:
- Font: 2 cách import font mới
- Flex, grid
- animation và transition (hiệu ứng)
- responsive (grid, flex, @media, đơn vị kích thước, max-width)
- CSS variable
- pseudo-class (:hover, :last-child,...)
- pseudo-element (::before, ::after) - sinh dữ liệu và luôn đi kèm thuộc tính CSS: content
- transform (2D, 3D)
- SCSS/SASS/BEM
